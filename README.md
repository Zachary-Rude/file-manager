# **Files: Manage files without Explorer.**

Files (not to be confused for every other file manager that is called Files) is a more advanced file manager for Windows, designed as a replacement for the deprecated Files app that is hosted at https://github.com/Zachary-Rude/files, and as a free and open-source alternative to Windows Explorer.
