# **Zach, Inc. Files: Manage files without Explorer.**

Zach, Inc. Files (not to be confused with every other file manager that is called Files) is a more advanced file manager for Windows, written in C# and designed as a replacement for the older, discontinued Zach, Inc. Files that was hosted at https://github.com/Zachary-Rude/files and written in VB.NET, and as a free and open-source alternative to Windows Explorer.

## __Features__
The new Zach, Inc. Files includes a whole bazooka of new features, many of which were not included with the old Files. They include:

* A menu bar
* The ability to create new files and folders from within the app
  * Can be done by either using *File &gt; New &gt; Text file* (for a new text file) or *File &gt; New &gt; Folder* (for a new, empty folder) within the main menu bar, or by using *New &gt; Text file* (for a new text file) or *New &gt; Folder* (for a new, empty folder) in the context menu
* The ability to copy, cut, rename, and delete files/folders
* An "Up" button (which, as you might expect, is the button with the icon showing a folder with an upwards-pointing arrow and automatically navigates to the folder directly above the current folder when it is clicked; can also be done via *Go &gt; Up* in the main menu bar or the `Alt` + `Up Arrow ↑` keyboard shortcut)
* An improved location bar that displays a folder icon on the far left side and requires users to press `Enter ↵` or `Return ↵` (rather than clicking a separate "Go" button) to navigate to the path that currently been typed into the location bar
* Different icons for different types of files
* The ability to choose the type of view used by the file/folder list from 5 available views: Large icons, Small icons, Details (which is the default), List, and Tiles
* The option to choose whether or not Files will show hidden files/folders in the file/folder list (the old Files, on the other hand, always showed hidden files and folders and did not offer any settings to disable this behavior)
* A clearable, dynamic recent folder list that can be accessed via *File &gt; Recent folders* in the main menu bar, and cleared via *File &gt; Recent folders &gt; Clear* in the main menu bar (but only when the list is not empty)
* The *Go to folder* feature, which opens up a window with a filesystem tree inside it where you can select various folders from your PC's filesystem and then automatically navigates to the selected folder when the "OK" button inside the window is clicked (can be accessed via *Go &gt; Go to folder* in the main menu bar)

...and many more.
